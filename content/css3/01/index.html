CSS3 가상클래스 :nth-child(N)
IE 9
1. 기본 숫자
괄호안에 5라는 숫자를 넣게 되면 5번째 자식요소를 가리키게 된다. 아주 간단하다. 
ul li:nth-child(5) {
    color: #ccc;
}
2. 키워드 : odd or even
“even”은 짝수의 순번(2, 4, 6…)을 가리키고, “odd”는 홀수의 순번(1, 3, 5…)을 가리킨다. 
ul li:nth-child(odd) {  
    color: #ccc;
}

tr:nth-child(even) {  
    color: #ccc;
}
3. 표현수 : an+b
표현수를 제대로 사용하기 위해서는 “N”과 “대수방정식”을 이해하여야 한다. “N”은 0부터 시작되고, 양의 정수(b)를 설정하면 방정식은 완성된다. 그래서 3n은 “3xn”이 되고, 양의 정수(b)에 3을 대입하면 전체 방정식은 “(3xn)+3″이 된다. 그렇다면, 0과 양의 정수를 대입하여 실례를 들어보자.

Ex) 3n+3이라는 표현수는 어떻게 적용이 되어질까? 
(3 x 0) + 3 = 3 = 3rd 요소
(3 x 1) + 3 = 6 = 6th 요소
(3 x 2) + 3 = 9 = 9th 요소
Ex) 2n+1이라는 표현수는 어떻게 적용이 되어질까? 
(2 x 0) + 1 = 1 = 1st 요소
(2 x 1) + 1 = 3 = 3rd 요소
(2 x 2) + 1 = 5 = 5th 요소
Ex) 여기서 “odd”와 “even”과 같은 방정식으로 “3n+0″ 또는 간단하게 “3n”으로 대신할 수 있다. 
(3 x 0) = 0 = no match
(3 x 1) = 3 = 3rd 요소
(3 x 2) = 6 = 6th 요소
(3 x 3) = 9 = 9th 요소
Ex) 이번에는 “+3″이라는 양의 정수 대신에 음의 정수를 사용해 보자. 그럼 4n-1은 어떻게 적용이 되어 질까? 
(4 x 0) - 1 = -1 = no match
(4 x 1) - 1 = 3 = 3rd 요소
(4 x 2) - 1 = 7 = 7th 요소

  ul li:nth-child(3n+3) {  
    color: #ccc;
  }

  ol li:nth-child(3n) {  
    color: #ccc;
  }

  tr:nth-child(4n-1) {  
    color: #ccc;
  }
4. CSS예제 
.example1 p:nth-child(3n+0) {
    background: red;
    color:#fff;
}
